# 🚀 快速开始 - 体验升级后的泡泡老大

## 最快体验方式（3分钟）

### 1. 打开游戏
```
双击 index.html
```

### 2. 创建测试单词本
点击"创建新单词本"，输入：
```
单词本名称：测试
单词1：cat - 猫
单词2：dog - 狗
单词3：bird - 鸟
单词4：fish - 鱼
单词5：lion - 狮子
```

### 3. 进入泡泡大战
1. 点击"泡泡大战"
2. 选择"测试"单词本
3. 点击"开始游戏"

### 4. 快速触发BOSS
连续正确拼写15个单词：
```
1. 看到"猫" → 输入 cat → 点击发射
2. 看到"狗" → 输入 dog → 点击发射
3. 看到"鸟" → 输入 bird → 点击发射
... 重复15次
```

### 5. 观察BOSS特性

#### 🔍 观察10秒
你会看到：
- ✨ BOSS会变大变小（呼吸效果）
- 🏃 BOSS会时快时慢移动
- 🎨 BOSS有眼睛和光环
- 📊 血条显示HP

#### 🎯 击败BOSS
1. 继续拼写单词
2. 发射子弹击中BOSS
3. 观察HP减少
4. 击中20次后...

#### 💥 震撼效果
- 三波巨大爆炸
- 屏幕震动
- 胜利提示
- +1000积分

## 💡 快速技巧

### 快速击败BOSS
1. **解锁激光弹**（1000积分）
   - 射线攻击，必中
   - 不怕BOSS移动

2. **解锁三炮台**（3500积分）
   - 三发齐射
   - 快速击败

3. **瞄准技巧**
   - BOSS变大时更容易击中
   - BOSS反弹时提前瞄准
   - 连续射击保持压制

### 快速获得积分
```
连击奖励：
1连击 = 150积分
2连击 = 200积分
3连击 = 250积分
10连击 = 600积分

技巧：不要失误，保持连击！
```
